## Testing & Verification: Gameplay Testing

### Objective
Students must thoroughly test game functionality to ensure collision detection, level flow, and character interactions work correctly.

### Evidence
Gameplay testing in custom levels:
- **GameLevelOutside and GameLevelMaze** - Complete level with collision, NPC interaction, transitions
- **All custom classes** - Testing collision detection and game mechanics
- **Player movement** - Testing keyboard controls and physics

### Implementation Details
- Manual play-through testing
- Testing all game mechanics
- Testing edge cases and boundary conditions
- Verifying collision detection accuracy
- Testing NPC interactions and dialogue

### Test Checklist
- Player movement responds to input
- Collision detection works correctly
- NPC interactions trigger properly
- Level transitions occur smoothly
- Win/lose conditions function correctly
- Score and stats update properly

### Key Concepts
- Manual functional testing
- Regression testing after changes
- Test coverage verification
- User acceptance testing
